# git
Información general sobre git...

## borrar
Comandos para borrar cosas en general.
git prune
### image
Comando para borrar una imagen:
git rm 

## limpiar
Comandos para limpieza de sistema.
### image
Comando para limpiar imágenes no usadas:
podman image prune
### container
Comando para limpiar contenedores detenidos:
podman container prune
